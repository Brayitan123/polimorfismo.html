Polimorfismo

Según Horstmann (2022):

"El polimorfismo es la capacidad de un objeto para tomar diferentes formas y comportarse de manera diferente según el contexto. Es uno de los principios fundamentales de la programación orientada a objetos que permite tratar objetos de diferentes clases a través de una interfaz común" (p. 189).

Martin (2019) amplía:

"El polimorfismo permite que referencias de tipos de clases base sean utilizadas para objetos de clases derivadas. En su forma más simple, permite que un mismo método actúe de manera diferente dependiendo del objeto sobre el cual actúa, proporcionando la capacidad de ejecutar el mismo comportamiento de distintas formas" (p. 143).
